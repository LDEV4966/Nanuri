# Nanuri Back-end

## Summary 

지속되는 코로나 19 상황으로 인해 인적 네트워크 형성이 어려운 상황에서, 교육 종사자 및 관련 업계 종사자들의 퇴직 후 사회활동이 줄어든 사람들에게
